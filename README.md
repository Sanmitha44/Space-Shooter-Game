# 🚀 Space Shooter Game

A fun browser-based Space Shooter game built using HTML, CSS, and JavaScript.

## 🎮 Features

- Move the spaceship using arrow keys
- Shoot enemies using the Spacebar
- Score tracking system
- Lives system (10 lives)
- High score tracking using Local Storage
- Game Over screen with final score
- Restart functionality

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 How to Play

- ⬅️ Left Arrow Key – Move Left
- ➡️ Right Arrow Key – Move Right
- Spacebar – Shoot Bullets

### Objective

Destroy as many enemies as possible and achieve the highest score before all lives are lost.
